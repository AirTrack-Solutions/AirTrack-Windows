User Guide

Home

The Home page shows the Total number of Airlines and Total number of Aircraft. It also has a dropdown to choose between filtering by Registration or Airlines.

  • If Registration is selected, you can enter full or partial registrations to search.
  • If Airlines is selected, a dropdown appears to let you choose an airline. Submitting will display all aircraft for that airline.
  • The Apply button will trigger the chosen search filter and update the results accordingly.
Airlines

The Airlines page shows a full list of all airlines in your database.

  • Airline names are listed in a table sorted alphabetically.
  • The list is static — airline names are not links and do not redirect.
  • You can edit or delete airlines using the action buttons on the right.
  • Airlines with no linked aircraft can still be added or modified.
  • This page is primarily for managing airline entries used throughout the system.
Add Airline

This page allows you to add a new airline to the system’s database.

  • Enter the full airline name and (optionally) a short ICAO or IATA code if known.
  • Click Add Airline to save the entry.
  • Once added, the new airline will appear on the Airlines page and will be available in the dropdown when adding or editing aircraft.
  • Airlines can be added ahead of time even if you haven't logged any aircraft yet.
Aircraft

The Aircraft page displays a full table of all aircraft currently in your database.

  • Each row includes the Registration, Flight Number, Type, Airline, Departure, Arrival, and Country of Registration.
  • You can edit or delete any aircraft using the buttons on the far right.
  • The page may support pagination for long lists of aircraft.
  • This view gives you a complete picture of your tracked aircraft.
Add Aircraft

This page lets you add a new aircraft to your database.

  • Fill in the Registration, Flight Number, Aircraft Type, and Airline at a minimum.
  • You can optionally add the MSN, Category, Country of Registration, Times Seen, Manufacture Year/Month, ICAO Address, and any notes.
  • The Spotted At, Departure, and Arrival fields track recent flight info.
  • Once submitted, the aircraft and an initial flight entry are both created in the system.
Reports

The Reports page offers a variety of prebuilt summaries and statistics:

  • Logged Airports: All airports seen in Departure or Arrival fields.
  • Most Seen Aircraft: Top 20 registrations with the most sightings.
  • Top Airlines: Most frequently seen airlines by total flights.
  • Most Frequent Routes: The busiest route combinations.
  • Top 10 Busiest Airports: Combined total of Departures and Arrivals.
  • First-Time Sightings: Most recent aircraft to appear for the first time.
  • Rare Airlines: Carriers with 2 or fewer appearances in your logs.
  • Top Countries: Based on Country of Registration.
  • Most Common Models: Aircraft types grouped by country.
  • Oldest Aircraft: Based on manufacture year.

You can explore any of these reports by clicking their name in the dropdown on the Reports page. All data is generated from your local database.

Upload Aircraft Image

This page allows you to upload an image for a specific aircraft already in your database.

  • Choose an airline from the dropdown. The list is sorted alphabetically.
  • Once selected, a second dropdown will populate with aircraft registrations for that airline.
  • Select the image you want to upload. Supported formats include JPG and PNG.
  • Click Upload to save the image to the server.

Note: If the registration is not already in your database, you’ll need to add it first using the Add Aircraft form.

Live Flights

The Live Flights link opens an external live aircraft tracking site (typically FlightRadar24) in a new browser tab.

  • The feature is provided as a convenience for real-time aircraft observation.
  • The external site is not affiliated with AirTrack, and access is subject to regional availability and external restrictions.
  • If the page fails to load, it may be due to location-based access limitations from your ISP or government.
  • The link opens with default coordinates (e.g., Western Sydney) and can be changed manually within the external site once opened.

If the site is blocked in your region, consider using a different tracking provider or check back later.

Export for Mobile

The Mobile Export feature generates a compact SQLite database (airtrack_mobile.db) for use with the AirTrack Mobile Android application.

  • What it exports: A mobile-optimised subset of your main AirTrack database.
  • Where it comes from: The file is generated on your server using the Export Mobile DB tool in Admin Tools.
  • Typical contents: Tables such as aircraft, reports, airlines, and airports.

How to use it with AirTrack Mobile:

  1. Download the generated airtrack_mobile.db file.
  2. Copy the file to your phone (for example into the Downloads folder).
  3. Open the AirTrack Mobile app.
  4. The app will automatically detect the file and ask if you want to import it.

Notes:

  • If the mobile app reports no such table, the export may be incomplete or outdated. Generate a new export and try again.
  • The database contains your personal logged sightings and will differ between users.
Admin

The Admin page includes tools for advanced management of the system and database. These actions affect stored data and should be used carefully.

  • Backup Database: Save a snapshot of the current database state.
  • Restore Backup: Restore a database backup file.
  • Flush Backups: Delete all existing backup files.
  • Flush Logs: Remove system log files.
  • Flush Database: Clear all aircraft, airline, and flight data.
  • Settings: Manage app-wide configuration (not available in Lite).
  • Launch Bug Tracker: Open the integrated bug reporting tool.
  • Find Aircraft Images: Trigger automatic search for missing aircraft images.
  • Check Airport Links: Scan airport entries for broken or outdated website and Wikipedia links.
  • Check Whitelist Links: Verify saved whitelist links for availability.
  • Clear Whitelist: Remove all custom whitelist entries from the database.

Please double-check actions like flushing data. They are irreversible.
