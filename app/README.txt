AirTrack Data Pack - Australia (CASA)

This data pack contains aircraft register data formatted for AirTrack Logbook.

Source:
CASA public aircraft register.

Notice:
AirTrack does not claim ownership of the original source data.
This pack is provided as a formatted convenience file for AirTrack users.
Users are responsible for ensuring their use complies with applicable laws and privacy obligations.

Import:
1. Open AirTrack Logbook.
2. Go to the aircraft data import area.
3. Import australia.sql into your AirTrack database.
